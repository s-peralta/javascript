Object Literal Notation
---------------------

```
var person = {
firstName:"John",
lastName:"Doe",
age:50,
eyeColor:"blue"
};

```

Object Constructor
---------------------

```
var Person = function(){
this.firstName = "John";
this.lastName = "Doe";
this.age = 50;
this.eyeColor = "blue";
}

```
